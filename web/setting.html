Chrome Secure Origin

Chrome may block keyboard input and some browser APIs on http://esp32mac.local because it is not HTTPS. Click the button to copy chrome://flags/#unsafely-treat-insecure-origin-as-secure, paste it into Chrome's address bar, add http://esp32mac.local to Insecure origins treated as secure, enable the flag, and relaunch Chrome.
